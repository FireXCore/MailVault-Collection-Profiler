# MailVault Collection Profiler `0.1.0-alpha.2`

This development alpha delivers the complete `0.1-C` physical inventory and read-only explorer
slice for MailVault schema-v3 archives.

## Highlights

- Windows desktop application and headless CLI.
- Read-only source preflight and consistent SQLite snapshot.
- Streaming message, MIME-part, relationship and blob inventory.
- Exact SHA-256 attachment occurrence reconciliation.
- Bounded physical availability and size inspection.
- Filename-history and exact-content relationship findings.
- Inventory search, stable cursor pagination, findings filters and object detail.
- Runtime evidence wrapper with SHA-256 manifest entries.
- GitHub-ready user, security, maintainer and release documentation.
- Sanitized product tour, CLI media and repository social preview.
- CI, CodeQL, Dependency Review, Dependabot and draft Windows release automation.

## Corrected release defects

- Windows snapshot durable sync now uses a write-capable handle.
- Source package npm lock no longer references an internal build registry.
- PowerShell quality gate is fail-fast and uses `npm.cmd` on Windows.
- Tauri progress channel no longer requires an unsupported `Debug` implementation.
- WiX/MSI version maps `0.1.0-alpha.2` to numeric `0.1.0.2`.
- Windows desktop compilation and snapshot regression tests are release gates.

## Installation

Download either the Windows x64 NSIS installer (`-setup.exe`) or MSI package. The installers are
unsigned in this alpha and can trigger Windows SmartScreen. Verify `SHA256SUMS.txt` before use.

## Known limitations

- Existing workspaces cannot be reopened through the desktop UI after restart.
- Pause/resume/cancel controls are not exposed.
- Full payload fixity and exact format identification are not active.
- Container expansion, JHOVE, OCR and semantic classification are not implemented.
- The canonical private 20–30 GB benchmark is not yet a published performance claim.

## Privacy

Do not attach real MailVault databases, profiler databases, EML files, attachments or unredacted
runtime evidence to issues.
